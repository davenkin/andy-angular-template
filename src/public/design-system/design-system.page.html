Icon

Icons can be added in two ways, one is to directly use PrimeNG icon ; the other is to use customized SVG icon. For customized SVG icon, make sure you register the SVG in icon-registry.ts before you use it.

SVG icon example

<app-icon icon="box"></app-icon>
Your can change the icon color using CSS color property, and change the font size using CSS font-size property.
